Команди, які використовуються при роботі з Git:

  • git add -A (додати всі внесені в файл зміни)
  • git commit -m "Commit message" (закомітити зміни. описати внесені зміни в імперативній формі, ніби надаючи вказівку гіту до їх впровадження)
  • git push origin HEAD (відправити всі зміни у віддалений репозиторій)
  • git pull (синхронізуватися з віддаленим репозиторієм. Стягнути "чужі" зміни, внесені в проект)
